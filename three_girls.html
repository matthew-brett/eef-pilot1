Three girls¶

  • Download this page as a Jupyter notebook (no outputs).

Now we have the machinery we need to solve the three girls simulation.

Let’s start with the problem of deciding whether the next child is a boy or a girl. We take a random number between 0 and 1, and if it is less than 0.5, we will call that 1, where 1 means a girl. Otherwise we will call it 0, where 0 means a boy.

We use the random module for that:

>>> import random
>>> type(random)
<class 'module'>

The random function in the random module, returns a random number between 0 and 1.

>>> random.random()
0.8899006007111...

Check what the function does with random.random? in a Notebook cell.

Now we’ll decide if the next child is a boy or a girl.

>>> # Get a random number, decide if it's a girl
>>> random_no = random.random()
>>> if random_no < 0.5:
...     # A girl
...     child = 1
... else:
...     # A boy
...     child = 0
...
>>> child
1

Run this a few times to make sure it produces 0 and 1 roughly the same number of times.

Make a family¶

Our task now is to make a for loop that collects four samples from get_0_or_1, and counts how many of these values are 1. This is our number of girls in the family.

>>> children = []
>>> for i in range(4):
...     child = get_0_or_1(0.5)
...     children.append(child)
...
>>> n_girls = children.count(1)
>>> n_girls

How many girls?¶

Now we take that code, and run it 10000 times. We collect the “number of girls” value in a list, for each time we go through the loop.

At the end, we calculate how many families had 3 girls. Divide by the number of families you made (10000).

Something like this:

>>> girl_counts = []
>>> for i in range(10000):
...     girl_count = girls_in_family()
...     girl_counts.append(girl_count)
...
>>> proportion = girl_counts.count(3) / 10000
>>> print("Estimate for probability of 3 girls is", proportion)

Run this a few times, to get an idea of the range of answers.
