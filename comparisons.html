Comparisons, True, and False¶

  • Download this page as a Jupyter notebook (no outputs).

We have seen expressions returning values that are numbers, and strings. Now we come across a new type of value, called boolean values.

These are values of True and False.

>>> a = 10
>>> # Is a less than 15?
>>> a < 15
True
>>> a < 8
False

We can test exact equality with double equals - ==:

>>> a == 15
False
>>> a == 10
True

We can test for inequality (not exactly equal) with != (read as “not equal”):

>>> a != 15
True
>>> a != 10
False
